Scope of AI Assistance

1. Layout & Visual Design System

2. Core Content Structuring

AI assistants were used as high-speed research companions to help organize the 21 NLP concepts and structure the comparative matrices. The final explanations, code implementation blocks, and references were reviewed, edited, and verified against standard academic guides (e.g., Jurafsky & Martin's Speech and Language Processing).

3. Interactive Code Blocks & Diagrams

Boilerplate Python scripts (utilizing nltk, spacy, sklearn, and transformers) were generated with AI assistance. The coordinates and connections for the SVG interactive nodes in the Workflow Library were mathematically calculated using AI assistance to ensure clean layout grids.
